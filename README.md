# Au Bout du Quai

Site web du restaurant gastronomique **Au Bout du Quai** — Les Sables-d'Olonne, Vendée.

## 🌐 Voir le site

Le site est accessible en ligne via GitHub Pages (voir Settings → Pages du dépôt).

## 📁 Structure

```
au-bout-du-quai-v2/
├── index.html          ← page d'accueil
├── chef.html           ← portrait du chef
├── carte.html          ← carte du restaurant
├── styles.css          ← styles partagés
├── reservation.js      ← modale de réservation Tableo
└── assets/             ← images et logo
```

## 🛠 Technologies

- HTML / CSS / JavaScript vanilla (sans framework)
- Polices Google Fonts : Playfair Display + Inter
- Système de réservation intégré : [Tableo](https://app.tableo.com/r/bGCK-09)

## ✏️ Modifier le site

Pour mettre à jour un texte ou une image, ouvrir le fichier `.html` correspondant
dans un éditeur de texte. Pour changer une photo, remplacer le fichier dans `assets/`
en gardant le même nom.
